User Commands

ChopBot commands are always prefaced by a dash (-). If this does not work, please contact the server staff of the Discord server you're trying to execute it on, as they may have changed it manually.

Command Description Aliases
-ping Pings the bot.
-help [command] Displays a list of commands, or extended help for a specified command. commands, helpme, call911
-info Provides bot information, such as a list of authors and libraries. details, what
-servers Links to all RT Family Discord servers. serv, rtfam
-report <user> <reason> Reports mentioned user to the mod team, reason required. Screenshots appreciated.
-join [user] [role] Allows you to join a role by name that's been specified as joinable by the server staff. Lists all joinable roles if none specified. addrole
-quote <message ID> [channel] Quotes a message by id. Mention channel if the message was not sent in the channel the command is being issued in.
-joindate [user] Displays your account creation date along with the date you joined the current server you're on. Other users can be specified to fetch their account creation and server join dates. me, myself
-8ball <question>? Magic 8-ball, does exactly what the toy does, memes included. 8, magic, mirror
-choose <query>, <query2> Makes a decision based off of the choices given. Can handle up to 100 queries per command. decide, choice
-coinflip Flips a coin. ðŸ™‚ for heads, ðŸ™ƒ for tails. coin
-roll [amount of dice] [type of dice] Rolls a die. Optionally specify the amount of dice and their type. dice, d20

Moderation Commands

Command Description Aliases
-userinfo [user] Displays user information. Returns your info if no other user is specified. whois, uinfo
-serverinfo Displays server information. sinfo
-sendmsg <channel/user> <message> Sends a message to the specified channel or user using ChopBot.
-purge [user] <amount> [all] Removes X amount of messages, optionally sent by Y user. Append the word 'all' to ignore role hierarchy.
-warn <user> <reason> Sends a formal warning to the mentioned user with the specified reason, and logs it. w
-mute <user> <reason> Mutes mentioned user and logs reason. m
-unmute <user> Unmutes mentioned user and logs it. um
-kick <user> <reason> Kicks mentioned user and logs reason. k
-softban <user> <reason> Kicks user from server and removes all their messages sent during the past 24 hours. sb
-ban <user> <reason> Bans mentioned user and logs reason. b
-vckick <user> <reason> Kicks user from voice if they are in a voice channel. vkick, vk
-vcban <user> <reason> Bans mentioned user from entering voice channels. vban, vb
-vcunban <user> Unbans mentioned user from server's voice channels. vunban, vub

Admin Commands

Command Description Aliases
-guide A guide on how to set the bot up for usage on your server.
-conf <set/get/reset/list/remove> [key] [value] Define per-server configuration via commmand.
