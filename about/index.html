Why make ChopBot?

Back in 2016, I (Rasmus) ran into the very specific issue where every bot that had good functionality for a larger server, also had its' fair share of issues. They could range anywhere from slow responses to bad user experience to extremely long periods of downtime. It was then I decided that I'd had enough and started working on what would one day become ChopBot.

The very first version of it was made using discord.py, a Python API wrapper for Discord. We now use discord.js, which is the same thing, but for JavaScript. At this time everything was still very experimental, mainly because I had never programmed anything before that.

Now, I knew very little about anything Python or JavaScript back then, so I asked Michael and Killian to help me out, which they were more than willing to. This was great, as they both brought knowledge and experience with them to the project. It's safe to say that without either of them, the project wouldn't be anywhere even close to where it is today.

Today, ChopBot is used on multiple Rooster Teeth family Discord servers (and one that's not!), providing moderation features not yet available without a bot, as well as logging for multiple things.
